Description

A parsec-based parser for the concrete syntax of the tutorial language, producing Tm and Ty values.

The concrete syntax supports:

  • Types: Unit/1, Void/0, Bool/2, t -> t, t * t, t + t
  • Terms: variables, lambda, application, pairs, injections, pattern matching, let, if/then/else, type annotations
  • Patterns: variables, unit, tuples, injections, annotated patterns
