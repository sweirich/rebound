Tutorial.Scoped.Eval

Description

Both evaluators operate on closed terms (Tm Z). The big-step eval returns a value or Nothing on an error; the small-step step returns Left Value when the term is already a value, Left Stuck when evaluation is blocked, or Right e' for the next term.
